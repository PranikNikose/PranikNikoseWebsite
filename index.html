// who I am

Builder by nature,
architect by habit.

I'm a full-stack developer based in Aurangabad, India, working across the full spectrum from React UIs to Spring Boot services. My day-to-day spans greenfield builds and legacy modernization — often both in the same sprint.

I have a deep interest in clean architecture: layered service design, DTO boundaries, proper exception handling, and code that doesn't fall apart six months later. Even in small projects, I think about how it scales.

Outside of work, I explore ideas like multi-tenant AI chatbot platforms, OTP-based 2FA flows, and experimental micro-apps like Button Roulette — a single mystery button that invokes a random Java class via reflection. Because why not.
