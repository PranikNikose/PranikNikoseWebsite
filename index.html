Button Roulette
A full-stack application built to demonstrate end-to-end software delivery, combining modern Java development with practical DevOps implementation. The application generates random outcomes through a Spring Boot backend while a React frontend provides an interactive user experience.
The project showcases complete ownership of the software lifecycle—from development and containerization to infrastructure provisioning, CI/CD automation, Docker image management, AWS deployment, domain integration, and automated application delivery using Infrastructure as Code principles.
▶ Live Demo GitHub →
ReactSpring BootJavaJava Reflection APIDockerDocker ComposeGitHub ActionsDocker HubTerraformAWS EC2AWS S3AWS DynamoDBElastic IPNginxCI/CD
Architecture Highlights
  • React Frontend — Single Page Application providing an interactive user interface and consuming backend REST APIs.
  • Spring Boot Backend — REST API responsible for request processing, roulette execution logic, and application endpoints.
  • Java Reflection API — Dynamically discovers and invokes spell implementations at runtime without hardcoded execution paths.
  • Dockerized Architecture — Frontend, Backend, and Nginx services are containerized and orchestrated using Docker Compose.
  • GitHub Actions CI/CD — Automated workflows build applications, create Docker images, publish images to Docker Hub, and deploy updates to AWS EC2.
  • Docker Hub Registry — Stores application images used by the automated deployment pipeline.
  • Terraform Infrastructure as Code — AWS infrastructure is provisioned and managed using Terraform with remote state storage in S3 and state locking through DynamoDB.
  • AWS Deployment — Application hosted on EC2 with Elastic IP configuration providing stable public accessibility.
  • Nginx Reverse Proxy — Routes incoming requests between frontend and backend services and exposes applications through a unified entry point.
  • Custom Domain Integration — DNS configured to expose the application publicly through roulette.praniknikose.top .
